Storage API 소개

sol studio 계정과 토큰만 있다면 무료로 1GB의 스토리지를 사용할 수 있습니다.

계정이 아직 없다면 얼른 회원가입하고 와주세요.

토큰이 아직 없다면 얼른 발급받고 와주세요.

파일 업로드

요청

URL http://sol-studio.tk/api/storage/upload
메소드 POST
요청 예시 
{
	"file": file, 
	"token":"nSP3uhFQ2TKi4XdfiV4coS5Od5KyUtRQzw937vGxAoi5Y1n8QIr52Kqp6HyVmqdfzTurL9S9T",
	"path":"/"
}




리턴

메시지 설명
Token Err 등록되지 않은 토큰입니다.
Path Err path가 "/"으로 시작하지 않거나, "../"이 포함되어있거나, 상위 폴더에 접근하려고 시도합니다.
File Err post요청에 파일이 존재하지 않습니다.등록하기
Limit Err 용량 한도를 초과했습니다.




Python 예제

import requests
files = open('{업로드될 파일 경로}', 'rb')
upload = {'file':files}
obj={"token":"{토큰}", "path":"{디렉토리, "/"으로 시작해야함}"}
res = requests.post('http://sol-studio.tk/api/storage/upload', files = upload, data = obj)
print(res.text) # Success라고 뜨면 성공입니다.








파일 다운로드

요청

URL http://sol-studio.tk/api/storage/download
메소드 POST
요청 예시 
{
	"token":"nSP3uhFQ2TKi4XdfiV4coS5Od5KyUtRQzw937vGxAoi5Y1n8QIr52Kqp6HyVmqdfzTurL9S9T", 
	"file":"/folder/file.txt"
}




리턴

메시지 설명
Token Err 등록되지 않은 토큰입니다.
Path Err path가 "/"으로 시작하지 않거나, "../"이 포함되어있거나, 상위 폴더에 접근하려고 시도합니다.
File Err 요청한 파일이 존재하지 않습니다.




Python 예제

import requests
obj={"token":"{토큰}", "file":"{파일 전체 경로, "/"으로 시작해야함}"}
with open("{다운로드될 파일 경로}", "wb") as file:
	res = requests.post("http://sol-studio.tk/api/storage/download", data = obj)
	file.write(res.content)








파일 탐색기

요청

URL http://sol-studio.tk/api/storage/explorer
메소드 POST
요청 예시 
{
	"token":"nSP3uhFQ2TKi4XdfiV4coS5Od5KyUtRQzw937vGxAoi5Y1n8QIr52Kqp6HyVmqdfzTurL9S9T",
	"path":"/"
}




리턴

메시지 설명
Token Err 등록되지 않은 토큰입니다.
Path Err path가 "/"으로 시작하지 않거나, "../"이 포함되어있거나, 상위 폴더에 접근하려고 시도합니다.
File Err 요청한 파일이 존재하지 않습니다.




Python 예제

import requests
obj={"token":"{토큰}", "path":"{탐색할 디렉토리, "/"으로 시작해야함}"}
res = requests.post('http://sol-studio.tk/api/storage/explorer', data = obj)
print(eval(res.text)) # 딕셔너리 형식으로 전달
